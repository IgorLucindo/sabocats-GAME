# Untitled Game
Platformer. constroy your own map and be the one to win the match.

# Setup
It is needed to have node.js installed.

# Execution
Execute the following command:
```
$ cd game/backend
```

```
$ nodemon server.js
```

Now enter the your ip adress and port 3000 on your browser. for example enter ``192.168.0.0:3000``
