# Criminal CATastrophe

Create your own map and be the one to win the match.

# Setup

It is needed to have node.js installed.

# Execution

To play game click in the following link:
* **Render:** [Link to Game](https://criminal-catastrophe-game.onrender.com)

# Dev

To open application locally, run the following command:

```
nodemon server/server.js
```

Now click on the following link to play game:
* **Localhost:** [Link to Game](http://localhost:3000)
